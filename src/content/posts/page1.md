# 铝热反应（Thermite Reaction）

## 1. 概述
铝热反应是**铝粉**与**金属氧化物**在高温下发生的剧烈**氧化还原放热反应**。
铝作为还原剂，夺取金属氧化物中的氧，生成对应的金属单质和氧化铝，并释放极高的热量。

## 2. 核心反应式
最经典的是氧化铁与铝的反应（铝热焊常用）：

**Fe₂O₃ + 2Al → 2Fe + Al₂O₃** （高温条件，ΔH < 0）

## 3. 反应特点
| 特性 | 描述 |
|:---|:---|
| **极高温度** | 反应瞬间温度可达 **2000~2500℃**，足以熔化生成的铁。 |
| **自维持性** | 一旦被引燃，反应会自行持续，无需外界供热。 |
| **强还原性** | 利用铝的强还原性置换出比铝不活泼的金属（如 Fe, Cr, Mn）。 |

## 4. 引发方式
该反应需要较高的活化能，通常使用**镁条**配合**氯酸钾**作为引燃剂：
1. 点燃镁条提供初始高温。
2. 触发铝热剂发生剧烈反应。

## 5. 主要应用
- **焊接**：铁路钢轨的无缝焊接（铝热焊）。
- **冶金**：冶炼高熔点金属（如铬、锰）。
- **军事**：早期用于制造燃烧弹和穿甲弹。

## 6. 安全警示
> ⚠️ **极度危险，非专业人员严禁尝试！**
- 反应产生强光和高温熔融物，必须穿戴防护装备。
- 需在空旷、无易燃物的环境中进行。

*总结：铝热反应的核心在于“以铝换铁，烈火熔金”。*
