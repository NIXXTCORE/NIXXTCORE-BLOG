# 镁热反应（Magnesiothermic Reaction）

## 1. 概述
镁热反应是以
**金属镁（Mg）**
为还原剂的冶金工艺。
由于镁的活性极强且价格相对低于钛、锆等金属，它常被用于还原卤化物或氧化物，以制取高熔点的稀有金属。

## 2. 核心反应式
最著名的应用是
**克罗尔法（Kroll Process）**
制取金属钛：

**TiCl₄ + 2Mg → Ti + 2MgCl₂** （高温，氩气保护）

其他常见反应：
- **SiO₂ + 2Mg → Si + 2MgO** （制取无定形硅）
- **2Mg + CO₂ → 2MgO + C** （镁在二氧化碳中燃烧）

## 3. 反应特点
| 特性 | 描述 |
|:---|:---|
| **强还原性** | 镁的电极电位很低，还原能力强，能夺取许多金属氧化物中的氧。 |
| **高温放热** | 反应剧烈放热，部分体系（如与卤化物反应）需严格控制温度以防爆炸。 |
| **环境敏感** | 镁极易与空气中的氮、氧反应，通常需要在惰性气体（如氩气）保护下进行。 |

## 4. 引发与应用
- **引发方式**：通常需要加热至镁的燃点（约 650℃）或在真空/惰性气氛中通电引燃。
- **主要应用**：
    - **生产钛/锆**：现代航空工业中钛金属的主要生产方式。
    - **制硅材料**：用于生产硅铁合金及半导体级硅材料。
    - **脱氧剂**：炼钢工业中用作脱硫剂和脱氧剂。

## 5. 安全警示
> ⚠️ **高危操作，严禁潮湿环境！**
- 镁粉易燃易爆，燃烧时发出**耀眼白光**并产生高温。
- 镁火**不可用水**扑灭（会加剧爆炸），必须使用 D 类灭火器或覆盖干沙。
- 反应产生的粉尘对呼吸道有强烈刺激作用。

---

*总结：镁热反应是利用镁的“泼辣”性格（强还原性），从化合物中“抢”出高价值金属的关键技术。*
