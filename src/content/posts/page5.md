# 碘‑镁法制备绝对无水乙醇（Magnesium‑Iodine Method）

## 1. 概述
95% 乙醇与水形成**共沸混合物**（78.15℃），普通蒸馏无法进一步除水。
实验室先以**生石灰（CaO）回流除水**制得 99.5% 无水乙醇，再以**镁屑 + 微量碘**去除残余微量水，制得纯度 >99.95% 的**绝对无水乙醇**。

> 碘的作用：破坏镁表面的 MgO 钝化膜，活化镁表面，促使镁与乙醇顺利反应生成乙醇镁。

## 2. 反应原理
**(1) 碘活化镁（引发反应）：**
Mg + I₂ → MgI₂（局部放热，去除氧化膜）

**(2) 镁与乙醇生成乙醇镁：**
Mg + 2C₂H₅OH → Mg(OC₂H₅)₂ + H₂↑

**(3) 乙醇镁与残余水反应（除水）：**
Mg(OC₂H₅)₂ + 2H₂O → Mg(OH)₂↓ + 2C₂H₅OH

→ 乙醇镁将痕量水不可逆地结合为不溶的 Mg(OH)₂，蒸馏即得绝对乙醇。

## 3. 仪器与试剂
| 类别 | 内容 |
|:---|:---|
| 仪器 | 干燥圆底烧瓶、球形回流冷凝管、无水 CaCl₂ 干燥管、蒸馏装置、接收瓶 |
| 试剂 | 99.5% 无水乙醇、洁净镁屑/镁条（砂纸打磨去氧化膜）、碘晶体（数粒） |
| 注意 | 所有玻璃仪器须**彻底烘干**，操作中严防空气中水汽侵入 |

## 4. 操作步骤
1. **装料**：250 mL 干燥圆底烧瓶中加 0.5～0.8 g 镁屑、10 mL 99.5% 无水乙醇，装回流冷凝管，上端加**无水氯化钙干燥管**。
2. **引发反应**：水浴微热至微沸→移去热源→**小心加入 2～3 粒碘（勿振荡）**。碘粒周围出现气泡、棕色褪去说明反应已引发；若无反应可稍加热或补加碘。
3. **完全反应**：待镁基本溶解（溶液变浑浊/乳白），再加入 90～100 mL 99.5% 无水乙醇和沸石。
4. **回流除水**：水浴加热**回流 30～60 min**，使乙醇镁充分与水反应。
5. **蒸馏收集**：改蒸馏装置，水浴加热蒸馏，**弃去前馏分（约 10 mL）**，收集 78～78.5℃ 馏分于干燥接收瓶，密封保存。

## 5. 关键注意事项
- ⚠️ **镁表面必须打磨干净**，否则难以引发。
- ⚠️ 所用无水乙醇含水量应 < 0.5%，含水量过高会导致引发失败。
- ⚠️ 反应初期产生 **H₂**，须远离明火，保持通风。
- ⚠️ 碘不可过量过多，否则可能生成碘代乙烷副产物。

---

*总结：碘‑镁法利用"镁→乙醇镁→与水反应除水"的思路，突破共沸限制，是实验室制备绝对无水乙醇的经典化学除水法。*
