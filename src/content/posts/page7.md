# 氯化铵与氢氧化钠反应制备氨水

## 1. 概述
利用**铵盐与强碱共热**放出氨气的原理，将 NH₃ 通入水中吸收，即可制得**氨水**。
实验室常用氯化铵（NH₄Cl）与氢氧化钠（NaOH）反应，装置简单、产气平稳，适合少量制备。

## 2. 反应原理
- **化学方程式：**  
  NH₄Cl + NaOH → NaCl + NH₃↑ + H₂O（加热）

- **离子方程式：**  
  NH₄⁺ + OH⁻ → NH₃↑ + H₂O

氨气极易溶于水（1 : 700），在水中形成氨水，部分电离：
NH₃ + H₂O ⇌ NH₃·H₂O ⇌ NH₄⁺ + OH⁻

## 3. 仪器与试剂
| 类别 | 内容 |
|:---|:---|
| **仪器** | 圆底烧瓶（或蒸馏烧瓶）、分液漏斗、双孔橡皮塞、导管、洗气瓶（可选）、锥形瓶/烧杯（作吸收瓶）。 |
| **试剂** | 氯化铵固体、氢氧化钠固体（或 30% NaOH 溶液）、蒸馏水。 |
| **防护** | 通风橱、耐碱手套、护目镜（氨气刺激性强）。 |

## 4. 操作步骤
1. **装料**：在干燥圆底烧瓶中加适量 NH₄Cl 固体，分液漏斗中加入浓 NaOH 溶液。
2. **组装**：按“固 + 液 → 气 → 液吸收”组装装置，检查气密性。
3. **反应**：缓慢滴加 NaOH 溶液，必要时用水浴微热，**控制滴加速度**以防剧烈喷涌。
4. **吸收**：将导气管插入装有冷水的吸收瓶液面附近（勿深入太多，防倒吸），用冷水浴冷却吸收瓶。
5. **停止**：先撤出吸收瓶中的导管，再停止加热，防止倒吸。

## 5. 注意事项
- ⚠️ **强腐蚀性**：NaOH 腐蚀皮肤与玻璃，操作需戴耐碱手套。
- ⚠️ **防倒吸**：氨气极易溶于水，停止反应前务必先将导管移出水面。
- ⚠️ **通风**：实验须在通风橱中进行，避免吸入高浓度氨气。
- ⚠️ **浓度控制**：若要较浓氨水，可减少吸收水量并加强冷却。

## 6. 氨水性质简述
- **外观**：无色透明液体，有强烈刺激性气味。
- **酸碱性**：呈碱性（pH ≈ 11–12），能使红色石蕊试纸变蓝。
- **不稳定性**：受热易分解：NH₃·H₂O → NH₃↑ + H₂O。

---

*总结：氯化铵与氢氧化钠共热是实验室制备氨气的经典方法，通过“碱夺铵之氢，放氨入水成氨水”，装置简便、反应可控。*
