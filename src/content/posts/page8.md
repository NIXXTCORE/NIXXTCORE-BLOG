# 双草酸酯与过氧化氢在乙酸钠催化下的荧光盛宴——过氧草酸酯化学发光

## 1. 概述
本实验模拟**荧光棒（Glow Stick）**的发光原理——利用**双草酸酯（如 CPPO 或 TCPO）**与**过氧化氢（H₂O₂）**在弱碱催化剂**乙酸钠（CH₃COONa）**作用下发生**过氧草酸酯化学发光（Peroxyoxalate Chemiluminescence）**，将反应释放的化学能转移给**荧光染料**，使其受激辐射发光。全程无热辐射，属于典型的**冷光（Cold Light）**现象。

## 2. 反应原理
**(1) 乙酸钠催化 H₂O₂ 生成亲核性 HOO⁻：**
CH₃COONa + H₂O₂ ⇌ CH₃COOH + HOO⁻（Na⁺）

**(2) HOO⁻ 亲核进攻草酸酯（以 CPPO 双(2,4,6-三氯苯基)草酸酯为例）：**
CPPO + HOO⁻ → 中间体 → 1,2-二氧杂环丁二酮（高能中间体）+ 2 C₆H₂Cl₃OH

**(3) 高能中间体分解，将能量转移给荧光染料（F）：**
1,2-二氧杂环丁二酮 → 2 CO₂ + 能量
能量 + F（基态）→ F*（激发态）→ F（基态）+ hν（可见光）

> 💡 发光颜色取决于所选荧光染料：  
> - **9,10-二苯基蒽（DPA）**→ 蓝光  
> - **荧光素（Fluorescein）**→ 绿光  
> - **罗丹明 B（Rhodamine B）**→ 橙红光

## 3. 仪器与试剂
| 类别 | 内容 |
|:---|:---|
| **仪器** | 干燥试管/比色皿、移液管、暗室或遮光盒、磁力搅拌器（可选） |
| **A液（草酸酯液）** | 双草酸酯（CPPO 或 TCPO，约 0.01–0.02 mol/L）溶于邻苯二甲酸二丁酯或乙酸乙酯 |
| **B液（氧化液）** | 30% H₂O₂ 与叔丁醇按 1:1 混合（增溶、减缓反应） |
| **催化剂** | 无水乙酸钠（微量） |
| **荧光染料** | 罗丹明 B / 荧光素 / DPA（乙醇或 DMF 稀溶液） |

## 4. 操作步骤
1. **配制 A液**：称取适量双草酸酯溶于有机溶剂（如邻苯二甲酸二丁酯），微热助溶，冷却备用。
2. **加染料与催化剂**：向反应容器中加入少量荧光染料溶液和一小撮（约 10–20 mg）无水乙酸钠，摇匀。
3. **加入 A液**：注入配制好的双草酸酯溶液，混匀。
4. **引发发光**：在暗处（关灯）快速加入 B液（H₂O₂/叔丁醇混合液），立即摇匀——溶液瞬间发出明亮荧光！
5. **观察**：可更换不同荧光染料，观察蓝/绿/红等不同颜色的化学发光。

## 5. 关键注意事项
- ⚠️ **避光操作**：荧光染料和部分草酸酯对光敏感，贮存在棕色瓶中。
- ⚠️ **H₂O₂ 浓度**：过高浓度会使中间体过快分解→发光强但持续时间短；过低则发光微弱。
- ⚠️ **乙酸钠用量**：只需微量；过量可能导致副反应或染料猝灭。
- ⚠️ **安全防护**：H₂O₂ 具腐蚀性，有机溶剂易燃，须在通风橱操作并佩戴护目镜。

## 6. 现象小结
混合后溶液本身不升温，但在暗处发出均匀、明亮的冷光——这正是商场荧光棒"一掰即亮"的化学本质：**化学能→电子激发能→可见光**，全程无热产生。

*总结：双草酸酯–过氧化氢–乙酸钠体系是过氧草酸酯化学发光的经典演示，被称为"化学烟花"，完美展现了化学能直接转化为光能的魅力。*
